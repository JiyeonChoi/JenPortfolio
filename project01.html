Safety Password Generator

Term: May 2015 ~ Jun 2015
Language: Java

  • Content

    “Safety Password Generator” is a program that helps reduce the loss of personal information leakage by making each site’s passwords unique but with regularities. In addition, it is a feature of this program to create a password that is encrypted but easy for the user to remember.
